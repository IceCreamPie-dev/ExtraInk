ExtraInk v2.0 API Reference

🌟 개요

ExtraInk v2.0은 Unity 기반의 고성능 비주얼 노벨 프레임워크입니다. SOLID 원칙을 기반으로 설계되어 확장성과 유지보수성을 제공하며, 메모리 풀링과 비동기 처리를 통한 최적화된 성능을 자랑합니다.
