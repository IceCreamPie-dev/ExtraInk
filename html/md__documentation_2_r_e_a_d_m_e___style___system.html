ExtraInk 스타일 시스템 사용법

개요

ExtraInk 스타일 시스템을 사용하면 다양한 대화 스타일과 캐릭터 애니메이션을 프리셋으로 관리할 수 있습니다.

구성 요소

1. DialogueStyle (ScriptableObject)

  • 대화 텍스트 및 애니메이션 설정을 포함하는 스타일 데이터
  • 다양한 프리셋 생성 가능

2. StyleManager (ScriptableObject)

  • 모든 스타일을 관리하는 중앙 관리자
  • 스타일 변경 및 적용 담당

3. StylePresets (Static Class)

  • 미리 정의된 스타일 프리셋들을 생성하는 유틸리티

설정 방법

1. StyleManager 생성

1. Project 창에서 우클릭
2. Create > ExtraInk > Style Manager 선택
3. Resources 폴더에 StyleManager 저장

2. 스타일 프리셋 생성

1. StyleManager 선택
2. Inspector에서 프리셋 생성 버튼 클릭
3. 자동으로 스타일 에셋 생성 및 추가

3. 기본 스타일 설정

1. StyleManager에서 Default Style 설정
2. Current Style이 자동으로 기본 스타일로 설정됨

스타일 종류

1. 클래식 스타일

  • 전통적인 비주얼 노벨 스타일
  • 타이핑 애니메이션 사용
  • 페이드 등장/퇴장 효과

2. 모던 스타일

  • 현대적이고 깔끔한 디자인
  • 페이드인 텍스트 애니메이션
  • 슬라이드 업/다운 캐릭터 애니메이션

3. 다이나믹 스타일

  • 활동적이고 생동감 있는 효과
  • 스케일 텍스트 애니메이션
  • 바운스 캐릭터 애니메이션
  • 텍스트 흔들림 효과

4. 미니멀 스타일

  • 간결하고 깔끔한 디자인
  • 애니메이션 없이 즉시 표시
  • 단순한 색상 구성

5. 레트로 스타일

  • 복고풍 클래식 게임 스타일
  • 느린 타이핑 애니메이션
  • 녹색 텍스트, 검은 배경

Ink 스크립트에서 사용법

스타일 변경

// 한국어
~ 스타일_변경("모던")
~ 스타일_변경("다이나믹")
~ 기본_스타일_복원()
// 영어
~ change_style("Modern")
~ change_style("Dynamic")
~ reset_to_default_style()

스타일 적용 캐릭터 등장/퇴장

// 현재 스타일의 기본 애니메이션 사용
~ 캐릭터_스타일_등장("주인공", "웃음", "center")
~ 캐릭터_스타일_퇴장("주인공")
// 영어
~ character_style_enter("hero", "smile", "center")
~ character_style_exit("hero")

일반 캐릭터 등장/퇴장 (효과 지정)

// 특정 효과 지정
~ 캐릭터_등장하기("주인공", "웃음", "center", "bounce")
~ 캐릭터_퇴장하기("주인공", "slide_left")

스타일 속성 설정

텍스트 애니메이션

  • Instant: 즉시 표시
  • Typing: 타이핑 효과
  • FadeIn: 페이드인 효과
  • SlideUp: 아래에서 슬라이드 업
  • SlideDown: 위에서 슬라이드 다운
  • Scale: 스케일 애니메이션

캐릭터 애니메이션

  • Fade: 페이드 인/아웃
  • Scale: 스케일 변화
  • Bounce: 바운스 효과
  • SlideLeft/Right/Up/Down: 슬라이드 애니메이션
  • Rotate: 회전 애니메이션
  • Flip: 플립 애니메이션

색상 설정

  • 대화 텍스트 색상: 일반 대화 텍스트
  • 화자 이름 색상: 캐릭터 이름
  • 강조 색상: 특별한 텍스트 강조
  • 배경 색상: 대화창 배경 색조

폰트 설정

  • 사용자 정의 폰트: 커스텀 폰트 사용 가능
  • 폰트 크기: 대화 텍스트 및 화자 이름 크기 조절

특수 효과

  • 텍스트 흔들림: 텍스트 진동 효과
  • 텍스트 웨이브: 텍스트 물결 효과
  • 사운드 효과: 타이핑 사운드 등

커스텀 스타일 생성

1. 스크립트로 생성

DialogueStyle customStyle = ScriptableObject.CreateInstance<DialogueStyle>();
customStyle.styleName = "커스텀 스타일";
customStyle.useTextAnimation = true;
customStyle.defaultAnimationType = TextAnimationType.Typing;
// ... 기타 설정
// 에셋으로 저장
AssetDatabase.CreateAsset(customStyle, "Assets/MyCustomStyle.asset");

2. 에디터에서 생성

1. Project 창에서 우클릭
2. Create > ExtraInk > Dialogue Style 선택
3. Inspector에서 원하는 설정 조정
4. StyleManager에 추가

주의사항

  1. Resources 폴더: StyleManager는 Resources 폴더에 위치해야 합니다.
  2. 스타일 적용 순서: 게임 시작 시 StyleManager가 자동으로 기본 스타일을 적용합니다.
  3. 에디터 전용: StyleManagerEditor는 에디터에서만 동작합니다.
  4. 플레이 모드: 스타일 테스트는 플레이 모드에서만 가능합니다.

트러블슈팅

StyleManager를 찾을 수 없음

  • StyleManager가 Resources 폴더에 있는지 확인
  • StyleManager 에셋이 올바르게 생성되었는지 확인

스타일이 적용되지 않음

  • DialogueManager와 CharacterManager가 Scene에 있는지 확인
  • 스타일이 StyleManager의 Available Styles에 추가되었는지 확인

애니메이션이 작동하지 않음

  • 해당 컴포넌트(RectTransform, SpriteRenderer 등)가 있는지 확인
  • 애니메이션 속도가 0이 아닌지 확인
